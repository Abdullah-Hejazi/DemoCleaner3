Democleaner2

Программа для обработки (перемещения/удаления) демо файлов, создаваемых при записи прохождений карт в Quake3.

Поддерживает множество опций:
- Возможность обрабатывать также подкаталоги
В самом начале в зависимости от типа предволагаемой обрабортки директории (с подкаталогами или нет) программа проходит по всем файлам и дальше определённым образом обрабатывает найденные файлы.

- очистка(удаление/перемещение файлов) может происходить как по лучшему времени на карте, так и по лучшему времени каждого игрока
- есть возможность выбора количества файлов с лучшим временем, которые требуется оставить.
(например можно сохранить по две лучшие демки каждого игрока)
- Можно обрабатывать mdf таймы как df, то есть при наличии записей двух типов, будет выбрана только лучшая по времени.
- несколько возможностей обработки записей с медленным временем:
Или удаление, или перемещение (всё в 1 папку), или пропуск, например для того, чтобы обработать только плохоименованые демки. 
 
Отдел "Demo mover"
Позволяет перемещать файлы в определённой папке по поддиректориям.
Актуально в больших каталогов файлов.

Главная его функция помимо простого перемещения файлов по алфавиту - также разделение записей в подкаталогах, и также разделение самих подкаталогов по подкаталогам!
В основном потому, что данная модификация игры не может отображать в списке выбора больше определённого числа записей (1024) и больше 64 каталогов.
Вследствие этого, для просмотра их приходится делить на подкаталоги, и, если это делается вручную, возникает много путаницы, а так же требует времени.
В основном, потому и была разработана данная программа, для организцации алгоритма распределения файлов.
- Также позволяет перемещать только записи определённого игрока

Дополнителльные опции относятся как к части с удалением файлов, так и к части с перемещением.
- обрабатывать все неправильноименованые записи (удялить, переместить, пропустить)
- Удалять пустые каталоги. После очистки или перемещения записей могут образовываться пустые каталоги, которые программа должна удалить.
- Удалять одинаково именованые файлы. То есть если в 2-х каталогах есть 2 одинаковые записи (проверяется только название, но не содержание), то программа оставляет только первую из них. Иначе - просто пропустит.
