Democleaner2

Программа для очистки демок
(Defrag демок, создаваемых при записи прохождений карт в Quake3)

Поддерживает кучу опций:
- Возможность обрабатывать также подпапки
В самом начале в зависимости от галочки Subfolders программа проходит по всем файлам, чсто найдёт или в рутовой дирректории, или в поддиректориях тоже. И дальше определённым образом обрабатывает найденные файлы.

- очистка может быть как по лучшему времени на карте, так и по лучшему времени каждого игрока
- есть возможность выбора количества лучших демок, которые оставлять
(например можно сохранить по две лучшие демки каждого игрока)
- Можно обрабатывать mdf таймы как df, то есть при наличии двух типов демок, будет выбрана только лучшая по времени.
- несколько возможностей обработки медленных таймов:
Или удаление, или перемещение (всё в 1 папку), или пропуск, например для того, чтобы обработать только плохоименованые демки. 
 
Отдел Demo mover
Позволяет распихивать демки в определённой папке по поддиректориям.
Актуально в больших каталогах демок

Главная его функция помимо просто таспихивания демок по алфавиту - также разделение демок в подпапках, и также разделение самих подпапок по подпапкам!
В основном потому, что дефраг не может отображать в списке выбора демок больше определённого числа демок (1024?) и больше 64 каталогов.
Потому для просмотра их приходится делить на подкаталоги, и путаться в них, если это делается вручную, да и муторно это.
В общем сложный алгоритм, но работает хорошо.
- Также позволяет перемещать только демки определённого игрока

Дополнителльные опции относятся как к клинеру, так и к муверу.
- обрабатывает все неправильноименованые демки (удялить, переместить, пропустить)
- Удалять пустые папки. После очистки или перемещения демок могут образовываться пустые каталоги, которые программа может удалять
- Удалять одинаково именованые файлы. То есть если в 2-х каталогах есть 2 одинаковые демки (проверяется только название, но не содержание), то программа оставляет только первую которую найдёт, иначе - просто пропустит.
